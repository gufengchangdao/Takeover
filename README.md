士气计算
1. 士气范围为0~1000，每个单位给一个基础士气值（例如500）
2. 单位攻击时+50士气，被攻击时-50士气
3. 士气>=1000进入英勇状态，士气<=0进入恐惧状态，持续10秒，然后士气回到200或者800
4. 士气<350进入低士气状态，士气>650进入高士气状态
5. 单位脱离战斗后士气逐渐趋于基础士气值
6. 士气值对不同单位的加成不一样，比如冲锋伤害、攻击距离、近战伤害、毒伤、冰伤

伤害计算
1. dmg = (dmgMin + Random.Range() * (dmgMax - dmgMin + 1))
2. 计算士气加成
3. 计算近战惩罚
4. 计算其他属性伤害
5. Mathf.FloorToInt(dmg)


保持阵型移动
1. 根据主单位当前坐标和目标节点计算一个路径节点列表
2. 主单位移动状态中会经过一个又一个节点，只要主单位到达了该节点就可以进行下一个节点了，每到达一个节点就把这个节点从路径列表中移除
3. 每个单位都有一个目标坐标TargetPos表示下一个节点的位置，Unit的Update里会不断增加移动冲量靠近这个坐标，TargetPos只在主单位开始下一个节点的时候给所有单位设置
4. 同时在移动过程中，主单位会不断对附近的单位施加冲量以此保持阵型，相当于单位的移动是两个冲量合并的结果
